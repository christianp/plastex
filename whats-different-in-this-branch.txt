This branch is used to collect patches used by Chirun (https://chirun.org.uk/),
before they're merged into the plasTeX main branch.

The intention is that this branch will frequently be rebased, as patches are
merged in.

At the moment, this branch includes:

* issue-365: tag amsmath equation environments
  (PR: https://github.com/plastex/plastex/pull/366) 
* issue-316: fix hyperref links
  (PR: https://github.com/plastex/plastex/pull/363)
* nocharsubs-eqnarray: don't apply charsubs inside eqnarray
  (PR: https://github.com/plastex/plastex/pull/310)
